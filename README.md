# Ardiuno-uno-fire-detector-


A simple fire detection system using Arduino Uno and a flame sensor. When fire is detected, a buzzer and LED alert the user.

---

##  Components

* Arduino Uno
* Flame Sensor
* Buzzer
* Breadboard & jumper wires

---

##  How It Works

The flame sensor detects fire and sends a signal to Arduino. Arduino then turns ON the buzzer and LED as an alert.

---

##  Pin Connections

* Sensor VCC → 5V
* Sensor GND → GND
* Sensor D0 → D2
* Buzzer + → D8
* Buzzer − → GND
